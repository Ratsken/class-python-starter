# Python Class Starter Program

This repository contains the starter project for our Python class.

## Project

- `django_auth/`: Django starter application we are building in class

## Getting Started

1. Create and activate a virtual environment:
   - PowerShell: `python -m venv venv` then `& .\venv\Scripts\Activate.ps1`
2. Install dependencies:
   - `pip install django`
3. Run the project:
   - `cd django_auth`
   - `python manage.py migrate`
   - `python manage.py runserver`

## Course Overview

This class is designed for beginners and builds toward practical software development skills. The first three weeks focus on core Python skills, then the extended curriculum introduces professional topics like OOP, web frameworks, testing, security, concurrency, and data science.

## Learning Objectives

By the end of training, learners should be able to:

- Write and explain Python fundamentals clearly

- Work with lists, tuples, dictionaries, and sets
- Build small practical applications
- Apply OOP concepts in Python
- Handle files, JSON/CSV data, and exceptions

## Core Glossary

- Syntax: Rules for writing valid code
- Bytecode: Intermediate instructions run by Python VM
- Variable: Named memory location for values
- Function: Reusable block of code
- Scope: Where a variable is visible
- Exception: Runtime error that can be handled
- List: Ordered mutable collection
- Tuple: Ordered immutable collection
- Dictionary: Key-value mapping
- Set: Unordered unique collection
- Module: Python file that can be imported
- Class: Blueprint for creating objects
- Object: Instance of a class
