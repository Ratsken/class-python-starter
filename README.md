# Python Class Starter Program

This repository contains the starter project for our Python class.

## Project

- `django_auth/`: Django starter application we are building in class

## Getting Started

1. Create and activate a virtual environment:
   - PowerShell: `python -m venv venv` then `& .\venv\Scripts\Activate.ps1`
2. Install dependencies:
   - `pip install django`
3. Run the project:
   - `cd django_auth`
   - `python manage.py migrate`
   - `python manage.py runserver`

## Notes

This is an in-progress starter codebase for class exercises and additions.
