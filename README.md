- `django_auth/`: Django starter application we are building in class

## Getting Started

1. Create and activate a virtual environment:
   - PowerShell: `python -m venv venv` then `& .\venv\Scripts\Activate.ps1`
2. Install dependencies:
   - `pip install django`
3. Run the project:
   - `cd django_auth`
   - `python manage.py migrate`
   - `python manage.py runserver`

## Python Programming Class Notes

Comprehensive training track covering Python fundamentals, control flow, functions, web development, data science, testing, and practical project work.

- Provider: Computer Pride Training Institute
- Duration: 3 to 15 weeks
- Schedule: Monday to Friday, 3 hours per day
- Contact Hours: 45 to 225 hours
- Approval: NITA APPROVED (TRN/966/VOL 1(17b))

## Course Overview

This class is designed for beginners and builds toward practical software development skills. The first three weeks focus on core Python skills, then the extended curriculum introduces professional topics like OOP, web frameworks, testing, security, concurrency, and data science.

## Learning Objectives

By the end of training, learners should be able to:

- Write and explain Python fundamentals clearly
- Use conditions, loops, and functions effectively
- Work with lists, tuples, dictionaries, and sets
- Build small practical applications
- Apply OOP concepts in Python
- Handle files, JSON/CSV data, and exceptions
- Use Git and GitHub for version control
- Build beginner web apps with Django and Flask

## Prerequisites

- No prior programming experience required
- Basic computer literacy (files, web browsing, text editing)
- Recommended machine: 4GB RAM, 5GB free disk space

## Weekly Learning Path

### Week 1: Introduction, Variables, and Data Types

- What Python is and why it is widely used
- Compiled vs interpreted languages and Python's hybrid model
- Environment setup, IDEs, and Git/GitHub basics
- Variables, naming conventions, numeric types
- Strings, slicing, formatting, and methods
- Booleans, operators, and user input

### Week 2: Logic and Control Flow

- Conditional logic: if, elif, else, nested conditions
- Loops: for, while, break, continue
- Collections introduction: lists and tuples
- Common list methods and operations

### Week 3: Functions and Advanced Structures

- Dictionaries and sets
- Defining and using functions
- Parameters, arguments, return values, and scope
- Practical assignment: simple contact manager

### Extended Curriculum (Variable Duration)

- Object-oriented programming (classes, objects, inheritance, polymorphism)
- File handling and persistence
- JSON and CSV workflows
- Exception handling and unit testing
- Security basics and token handling
- Threading and multiprocessing
- GUI development with Tkinter
- Data science with NumPy and Pandas
- Web development with Django and Flask

## Core Glossary

- Syntax: Rules for writing valid code
- Bytecode: Intermediate instructions run by Python VM
- Variable: Named memory location for values
- Function: Reusable block of code
- Scope: Where a variable is visible
- Exception: Runtime error that can be handled
- List: Ordered mutable collection
- Tuple: Ordered immutable collection
- Dictionary: Key-value mapping
- Set: Unordered unique collection
- Module: Python file that can be imported
- Class: Blueprint for creating objects
- Object: Instance of a class
